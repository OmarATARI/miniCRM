### Stack

- Front: React JS
- Database: Firebase
- Node & npm
- Testing: Jest & Enzyme

### Get started

Build and run the app using :

`$ docker-compose up --build -remove-orphans`

Launch tests :

`$ docker-compose exec client npm test`


The App should be App :

visit client : http://localhost:3000


To check the status :

`$ docker-compose ps`
